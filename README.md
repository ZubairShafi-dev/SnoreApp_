# SnoreApp_
💤 Snore Tracker – AI Powered Sleep Monitor

Snore Tracker is an AI-powered Android application designed to monitor, detect, and analyze snoring patterns during sleep. The app helps users understand their sleep quality by identifying snoring events in real time and presenting detailed insights in a clean, modern UI.

📱 About the App

Snore Tracker uses a custom TensorFlow Lite (TFLite) audio model to intelligently detect snoring sounds and distinguish them from normal breathing or background noise.

During sleep, the app continuously listens in the background and analyzes audio signals to:

Detect snoring events

Measure intensity, frequency, and duration

Record and store snore audio samples

Generate a detailed sleep quality report every morning

The app also provides visual analytics and audio playback, allowing users to review their sleep patterns and make informed improvements for healthier and more peaceful sleep cycles.

✨ Key Features

🔊 Real-time snore detection using on-device AI

🧠 Custom TFLite audio classification model

📊 Visual sleep insights (graphs & statistics)

🎧 Snore audio playback for review

🌙 Daily sleep summary reports

⚡ Fully offline-capable detection (privacy-focused)

🎨 Modern dark-themed UI optimized for night usage

🛠️ Tech Stack

🤖 TensorFlow Lite – Custom audio classification model

🎙️ Android Audio Recorder – Real-time audio capture

📐 Signal Processing – Feature extraction & noise handling

🧩 Kotlin + Jetpack Components – Clean Android architecture

🗄️ Room Database – Local storage for sleep & snore data

☁️ Firebase – Analytics / optional cloud features

🧱 Architecture

The app follows modern Android architecture principles:

MVVM pattern

Separation of UI, data, and business logic

Lifecycle-aware components

Efficient background processing for long-running sleep sessions

🎯 Use Cases

People suffering from snoring or sleep disturbances

Users who want to analyze sleep quality

Health-focused users tracking nightly sleep behavior

Demonstration of AI audio processing on Android

🔒 Privacy First

All audio processing happens on-device using TensorFlow Lite.
No raw sleep audio is uploaded without user consent.
